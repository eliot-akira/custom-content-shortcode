Mobile detect

Enable the mobile detect module under Settings. The following shortcodes are used to display content based on device type. [is_phone] [isnt_phone] [is_tablet] [is_mobile] [is_computer] [isnt_computer] The [is_mobile] shortcode includes both tablet/phone, and [is_computer] is for all other devices, for example, laptops and PCs. Using these in combination, you can selectively load images, page sections, stylesheets, etc. You can even redirect users to another URL.

Show differently sized images

[is_mobile]     <img src="[url uploads]/birds-150x150.jpg"> [/is_mobile] [is_computer]     <img src="[url uploads]/birds-600x600.jpg"> [/is_computer]

Load stylesheets according to device type

[is_computer]     [load css="style.css"] [/is_computer] [is_mobile]     [load css="style-mobile.css"] [/is_mobile]

Redirect mobile visitors to a different location

[is_mobile]     [redirect][url site]/mobile/[/redirect] [/is_mobile] These last two would be placed in a custom field named css, to load in the header of the page.

Body class

There are CSS classes added to the <body> element, for styling purposes: .is_phone, .isnt_phone, .is_tablet, .is_mobile, .is_computer and .isnt_computer.

Library

The detection algorithm is based on a lightweight PHP class, Mobile Detect version 2.8.12.
