User status

The [is] shortcode is used to display content if the user is a specific user, an admin, non-admin, or logged in/out.

Display user status

[is admin]You are an administrator.[/is] [is login]You are logged in.[/is] [is logout]You are logged out.[/is] [is user="john"]You are John.[/is] Available parameters:
  • admin - administrator
  • login - logged in
  • logout - logged out
  • user - user name or ID - multiple values are possible: "1,3,22"
You can also use the [isnt] shortcode, to see if the user is not admin, etc.

Login and logout

The [url] shortcode is also used to display login and logout links.

Display a login link

<a href="[url login]">User login</a>

Display a logout link with redirect to home

<a href="[url logout go='home']">Logout</a> Available parameters:
  • login - login link
  • logout - logout link
  • go - redirect user after login/logout
For the redirect link, you can either use a full URL (www.example.com) or the name/slug of any post or page. The slug home will return the user to the home page of your site. Here is an example using both [is] and [url] to show a login/logout link based on user status. [is logout]     <a href="[url login go='home']">         Login Button     </a> [/is] [is login]     <a href="[url logout go='somewhere']">         Logout Button     </a> [/is]

Redirect users

To redirect users to a different URL according to user status, device type, etc., use the [redirect] shortcode. As soon as it is executed, the user will be taken to another page.

Redirect if visitor is not logged in

[isnt login]     [redirect]http://example.com/guest/[/redirect] [/isnt] Please note that the destination URL is not a parameter, it is wrapped inside. This is to allow relative links such as: [is login]     [redirect][url site]/user-area/[/redirect] [/is] The [redirect] shortcode can be placed anywhere, but the best place is in a custom field named css, so it can run at the beginning of the page.

Comments

For displaying comments of the current post, or in a query loop, use the [comment] shortcode.

Display comment count, input form, or comments list

Comment count: [comment count] Comment input form: [comment form] Comments list: [comment template] or [comments] You can also display the total comment count after a loop: Total number of comments: [comment total] By default, the comments list is displayed by comments.php in the theme directory. If you want to specify a different template: [comment template="/short-comments.php"] Here is an example of a comments list that shows when clicked, using Bootstrap accordion. <a class="accordion-toggle" data-toggle="collapse"     data-parent="accordion" href="#comments-area">         Comments ([comment count]) </a> <div id="comments-area" class="panel-collapse collapse">     [comments] </div>
