Gallery field

Enable a simple gallery field for any post type, under Settings -> Gallery Fields.
You can add, order, edit details and remove images from the field.

Display images from the gallery field

[attached gallery]     Title: [field title]     Full-size image: [field image]     Caption: [field caption] [/attached] For each image, these fields can be displayed: id, title, image, image-url, caption, description, thumbnail, and thumbnail-url.

In a Loop

Display gallery fields of each post in a loop

[loop type="post" category="colorful"]     Post Title: [field title]     Description: [content]     [attached gallery columns="4"]         [field thumbnail]     [/attached] [/loop]

Native or Bootstrap gallery

You can display all images in the gallery field using a native gallery or Bootstrap carousel.

Display a native gallery or Bootstrap carousel

[content gallery="native"] [content gallery="carousel"] You can pass the following parameters to the native gallery: orderby, order, columns, size, link, include, exclude. For details, please refer to the Codex: Gallery Shortcode.

Individual image

The [content] shortcode can display individual images of the gallery field.

Display the 3rd image in the gallery field

[content field="gallery" num="3"] ..or.. [field gallery num="3"]

Group

When using the [loop] to generate multiple Bootstrap carousels, the following will put images from each post in its own carousel. [content gallery="carousel" group="gallery-{ID}"]
