Overview

Main features

  • The [loop] and [content] shortcodes
  • View the site's content structure

Included

  • Gallery field for any post type
  • Cache the result of query loops - New!
  • Display comments and attachments
  • User info and content based on user status
  • Load HTML/CSS/PHP/JavaScript files
  • Relative URLs for links and images

Support for other plugins

  • Advanced Custom Fields - Image, relationship, gallery, repeater, flexible content
  • WCK Fields and Post Types - Text, select, checkbox, radio, upload, repeater
