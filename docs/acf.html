Advanced Custom Fields

Relationship

Use the [related] shortcode to loop through a relationship field.

Display titles and thumbnails of related posts

[related field="relationship"]     [field title]     [field thumbnail] [/related] You can use the same shortcode for a Post Object field.

Repeater field

Use the [repeater] shortcode to loop through a repeater field.

Display repeater fields

[repeater field="boxes"]     [field title]     [field image="image"]     [field description format="true"] [/repeater] For an image field, use the image parameter to display the field. You can set the size parameter to: thumbnail, medium, large - default is full-size. If the image field returns as URL, set in="url". To display a repeater inside a repeater, use [-repeater]. Please note that the inner repeater field must have a different name than its parent.

Flexible content field

[flex field="flexible_content"]       [layout name="title_text"]         [field title]         [field text]     [/layout]       [layout name="title_image_text"]         [field title]         [field image="image" size="thumbnail"]         [field text]     [/layout]       [layout name="gallery"]         [acf_gallery field="gallery"]             [acf_image size="thumbnail"]         [/acf_gallery]     [/layout]   [/flex]

Gallery

For gallery fields, use the [acf_gallery]  shortcode.

Display images with title

[acf_gallery field="images"]     [acf_image]     [acf_image field="title"]     [acf_image size="thumbnail"] [/acf_gallery] The [acf_image] shortcode displays each image in the gallery field. It can also display these fields: id, title, caption, alt, url, and description. You can set the size parameter to: thumbnail, medium, large. The default is full-size.
You can pass the image IDs to another shortcode.

Pass the images to another shortcode

[pass acf_gallery="images"]     [isotope_gallery ids="{FIELD}"] [/pass]

ACF fields in other posts

Display gallery, relationship, repeater, or flexible content fields from other posts inside the loop. [loop name="hello-world"]     [repeater field="boxes"]         [field title]         [field image="image"]         [field description format="true"]     [/repeater] [/loop]

Columns

You can use the columns parameter for gallery, repeater, or flexible content. For details, please see its description for [loop], under Other Parameters.
